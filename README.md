# BowlScan – Cricket Bowling Biomechanics Analysis

BowlScan is a computer vision system that analyzes fast bowling actions using pose detection and biomechanical metrics.

Features
- ICC elbow legality detection
- Bowling action classification
- Injury risk analysis
- Hip–shoulder separation tracking
- Trunk flexion analysis
- Automatic PDF reports

Technologies
- Python
- MediaPipe
- OpenCV
- Streamlit
- NumPy / SciPy
- Matplotlib

How to Run

pip install -r requirements.txt

streamlit run streamlit_app.py
